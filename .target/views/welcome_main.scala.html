Pacemaker is a Web application running on the play framework with CSS styling from Sematnic UI

The purpose of this application is to keep track of a user's activities. Currently a user can create an activity by giving the activity's Date, Time, Duration, Type (Run, Walk, Go grocery shopping etc.) The location at which it is taking place (e.g. meeting at the bank) and the distance at which the activity will last or the distance to where the activity is. Once an activity is created it can be viewed from the user's Dashboard. From the Dashboard the user can add a location (latitude and longitude) to an activity. A user can add multiple locations to an activity and this is the Route part of the activity. When the latitiude and longitude are added, they should be clickable and bring the user to the location of the latitude and longitude on Google Maps.
